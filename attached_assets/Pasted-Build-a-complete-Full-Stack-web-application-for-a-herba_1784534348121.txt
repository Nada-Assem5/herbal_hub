Build a complete Full Stack web application for a herbal gummies brand called "Pure Botanica".

## Goal

The website helps parents determine which herbal gummies are most suitable for their child by answering an interactive assessment.

Maximum child age is 12 years old.

The application must include both Frontend and Backend.

Use:

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Hook Form
- Zod Validation
- Prisma ORM
- PostgreSQL (Supabase)
- API Routes
- Server Actions where appropriate

Assessment Rules

Recommendation Logic

Product 1

Focus Gummies

Recommended when score indicates:

- Hyperactivity
- Poor concentration
- Easily distracted
- Restlessness

Benefits

• Supports focus
• Helps maintain calm behavior
• Herbal ingredients

--------------------------------

Product 2

Mineral Gummies

Recommended when:

- Poor nutrition
- Low vegetable intake
- Daily nutritional support

Benefits

• Natural minerals
• Daily wellness
• Healthy growth

--------------------------------

If both scores are high,
recommend both products.

Show explanation for each recommendation.

--------------------------------

Scoring System

Focus Score

Hyperactivity = +3

Difficulty focusing = +3

Can't sit still = +2

Homework difficulty = +2

Mineral Score

Poor diet = +3

No vegetables = +3

Needs daily nutrition = +2
